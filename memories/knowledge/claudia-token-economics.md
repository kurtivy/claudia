## What
Pump.fun launched automated agent buybacks (Mar 2026) connecting agent revenue to token burn. Auto.fun competes with ecosystem-flywheel model. Binance AI Pro commoditizes agentic trading at $9.99/mo. $CLAUDIA needs differentiation beyond "AI trades."

## Why It Matters
$CLAUDIA is a pump.fun agent token with buyback eligibility. The buyback mechanism is the clearest path to connecting Claudia's real utility to token value, but only if revenue exists.

- **Pump.fun buyback mechanics**: Developer sets % of agent revenue for buybacks. Agent earns SOL/USDC, that portion auto-buys and burns token. $10 minimum threshold. Existing tokens (bonding curve or PumpSwap) eligible.

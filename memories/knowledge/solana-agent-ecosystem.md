# Solana Agent Ecosystem -- March 2026

## Infrastructure

### Solana Developer Platform (SDP)
- Launched March 24, 2026 at Digital Asset Summit NYC
- AI-ready infrastructure aggregating ecosystem tools for institutional deployment
- Mastercard and Western Union named as users for tokenized assets and payment flows
- APIs for payments, tokenized assets, compliance
- Machine-readable skill file at solana.com root: agents self-teach wallet creation, transactions, on-chain programs without documentation

### Agent Registry
- 9K deployed agents on Solana (identity + reputation protocol)
- Registration required for participation in ecosystem apps (e.g. Babylon prediction game)

### x402 Agentic Payments
- Solana processes 65% of all agentic payments via x402
- 35M transactions, $10M volume (Coinbase-built, Cloudflare joined)
- 15M on-chain agent payments processed on Solana specifically
- Base + Solana = 97% of agent-to-agent transaction share
- Stablecoins becoming default payment rail for AI compute/services ($650B monthly stablecoin volume on Solana)

### Identity-Payment Convergence
- ERC-8004 + x402 emerging as canonical stack
- ERC-8004 = "should agents transact" (trust layer), x402 = "how agents pay" (mechanics layer)

### MoonPay Open Wallet Standard (March 23)
- Open-source agent wallets across 8 chain families
- Backed by PayPal, Ethereum Foundation, Solana Foundation, Circle, OKX, Ripple, Base, Polygon, Sui, Arbitrum
- AES-256-GCM encrypted keys, never exposed to agent process
- Available on GitHub/npm/PyPI
- Kurt has pending decision on OWS integration

## Agent Platforms

### ElizaOS
- Dominant agent framework: 17,600+ GitHub stars, 5,300+ forks, 1,350+ contributors, 200+ plugins
- Functions as "Linux layer" for on-chain AI agents
- Unified message bus across X, Discord, Telegram, on-chain
- Supports native wallet control, multi-agent swarms
- Adopted Chainlink CCIP for cross-chain agents

### TARS
- Solana Foundation-backed AI Market
- Aggregates agent frameworks: Sona, Akira, Eliza, Rig, Solana Agent Kit
- One-click agent deployment + instant tokenization via bonding curves
- 892M/1B token supply circulating
- Bonding curve tokenization uses same model as pump.fun

### Virtuals Protocol
- $479M total aGDP, $2.67M total agent revenue, 1.78M jobs completed, 23,514 unique active wallets (Feb 2026)
- Separate stat (possibly different methodology): 17K+ agents, $39.5M revenue
- Tracks "aGentic GDP" -- the emerging metric for agent economic output

### Babylon
- Multiplayer prediction game: humans and AI agents compete on same feed/leaderboard
- Agents register via Solana Agent Registry

## Key Predictions (DAS, March 25)

Vibhu Norby (Solana Foundation CPO):
- "99.99% of all onchain transactions in 2 years will be driven by agents, bots, and LLM-based wallets"
- **Assessment**: Marketing, not analysis. Solana is already >90% bot transactions if you count MEV/arbitrage. The real question is economic value, not volume. The 15M payment number is more honest than the 99.99% claim.

## Narrative vs Reality Gap

**What's real:**
- Agent Registry exists and functions (identity + reputation)
- ElizaOS has genuine developer adoption (17.6K stars)
- Pump.fun Tokenized Agents feature is live
- x402 protocol has real volume (35M txns, $10M)
- Stablecoin infrastructure is ready ($650B monthly)

**What's not real yet:**
- Autonomous agents generating meaningful revenue at scale
- Agent-to-agent commerce at scale
- Any agent maintaining itself without human intervention for extended periods
- "Agentic GDP" described as "moving from theory to measurement" in Feb 2026 ecosystem report -- they can't measure it yet
- 454 hackathon submissions shows developer interest, not production deployment
- Most "agents" are cron jobs with wallets

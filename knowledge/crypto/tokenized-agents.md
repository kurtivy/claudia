# Tokenized Agents — Knowledge Node

_Last updated: 2026-03-24_

## The Thesis

AI agents that earn their own on-chain revenue are structurally different from agents dependent on their creator's wallet. Tokenization creates alignment: token holders benefit when the agent performs. The agent has skin in the game.

Core mechanic: agent earns revenue → buys back its own token on the open market → burns the tokens. Supply shrinks. Existing holders benefit. This is the same logic as share buybacks but on-chain and autonomous.
